# Patient Health Database

Author: Mimi Larrieux

## Purpose
The code in this repository will run a patient health database.
The code is written in python.

## Specifications
The database will contain:
* Patient Name
    + First Name
    + Last Name
* Age
* Patient ID
* Test Results
    * Test name
    * Test Result
    1. HDL
    1. LDL
